# 图片资源说明

## Hero 背景图片

将你的背景图片命名为 `hero-bg.jpg` 并放在这个文件夹中。

### 推荐规格：
- **分辨率**: 1920x1080 或更高
- **格式**: JPG 或 PNG
- **文件大小**: 建议压缩到 500KB 以内
- **内容**: 科技感、网络、团队相关的图片

### 图片建议：
1. 可以使用服务器机房、网络设备的照片
2. 团队合影或活动照片
3. 抽象的科技背景图
4. 确保图片亮度适中，文字能清晰显示

### 其他可选图片：
你也可以使用其他名称的图片，只需修改 `src/views/Home.vue` 中的路径：
```css
background-image: url('/images/你的图片名.jpg');
```

## 其他图片

如果需要在其他地方使用图片，也可以放在这个文件夹中：
- `logo.png` - 网站Logo
- `team-*.jpg` - 团队成员照片
- `event-*.jpg` - 活动照片
- `blog-*.jpg` - 博客封面图

## 临时占位符

如果暂时没有图片，可以使用在线占位图服务：
- https://picsum.photos/1920/1080
- https://placeholder.com/1920x1080

或者继续使用现在的渐变色背景（已设置为后备方案）。
